# love_chaoxing
chaoxing学习堵签到
1.如何运行：解压缩，进入解压的文件夹，双击run.bat，在弹出的超星本地服务窗口输入校验码，登录页面会自动弹出，弹出的两个命令行窗口，完成签到之前不要关闭
2.如何登录：输入某通账号密码，正常登录即可。登录失败可以尝试先网页版或者手机版登录，再重试，也可点击cookie登录，按提示操作
3.如何进入签到页面：先点击对应课程，再选择对应签到
4.新签到不显示：在下方点击课程，重复第三条
5.普通签到：点击即可
6.图片签到：可以尝试点击一键借用同学图片，该功能未经过测试，不保证能正常使用
7.位置签到：建议使用一键获取老师位置，也可提前记住教室坐标，自己输入，尽量不用地图自选位置，高德的api默认中心位置在北京，很难用。如果显示未获取老师位置，大概率是老师没设置签到范围，用学习通直接签试试
8.签到码签到和手势签到，点击一键查看签到码，其他按键都是摆设
9.二维码签到：不支持绕过，但可以生成签到链接，将链接发给好友，好友打开链接，点击扫码，即可完成代签。操作方法，二维码签到一定要在名为超星签到助手的命令行窗口 找到白框四面框起来的链接，在浏览器打开该链接，在这个页面里找到二维码签到页面，点击发链接送给手机，才能生成可被外部访问的链接，发送的链接里附带姓名，电话号码，超星账号密码等敏感信息，一定谨慎使用，托付给可靠的朋友
此程序依赖cloudflared实现内网穿透，run.bat有内置下载，若无法自动下载，请自行下载cloudflared amd64 版本到本文件夹，并改名cloudflared.exe
